## props :

| 属性        | 描述                         | 默认值 | 可选值                   |
| ----------- | ---------------------------- | ------ | :----------------------- |
| imgList     | 图片内容                     | -      | ['xxxx.jpg',,'xxxx.jpg'] |
| autoPlay    | 是否自动播放                 | false  | true/false               |
| playTime    | 自动播放的时间               | 2 (s)  | 正整数                   |
| keysControl | 是否监听键盘左右按钮切换事件 | true   | true/false               |

## slots :



| 插槽名称 | 描述       | 默认值 | 备注     |
| -------- | ---------- | ------ | :------- |
| leftBtn  | 自定义内容 | ←      | 手动翻页 |
| rightBtn | 自定义内容 | →      | 手动翻页 |
|          |            |        |          |
